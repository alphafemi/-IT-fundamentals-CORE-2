## 1. Accessing the Control Panel

| Method | Action |
|--------|--------|
| Search | Type `control` → Select Control Panel |
| Run | `control.exe` |

**Recommended view for exams:** **Large icons** or **Small icons** (shows all applets alphabetically, not categorized)

## 2. Key Control Panel Applets

### Internet Options

| Tab | What It Configures |
|-----|-------------------|
| **General** | Browsing history, appearance (colors, fonts, languages), tabs |
| **Security** | Security levels by zone (Internet, Local intranet, Trusted sites, Restricted sites) |
| **Privacy** | Cookies, pop-up blocker, site permissions |
| **Content** | Certificates, AutoComplete |
| **Connections** | Dial-up, VPN, proxy server settings (LAN Settings) |
| **Programs** | Add-ons, default programs for links |
| **Advanced** | Detailed protocol settings, accessibility, security options |

> **Exam Tip:** Proxy server configuration is found under **Connections tab → LAN Settings**.

---

### Devices and Printers

**Purpose:** View and manage all connected devices (local and network)

**Features:**
- User-friendly icons for printers, microphones, etc.
- Right-click any device → Configure or change properties
- Easier to use than Device Manager for basic tasks

---

### Programs and Features

| Function | Description |
|----------|-------------|
| **Uninstall** | Remove installed applications |
| **Change/Repair** | Modify application installation |
| **Turn Windows features on/off** | Enable/disable built-in Windows components (e.g., IIS, Media Features) |

**Information shown:** Application name, installation date, size on drive

---

### Network and Sharing Center

**What you can do:**
- View active network connections (wired/wireless)
- Click connection → View status → Properties
- Configure IPv4/IPv6 settings (IP address, subnet mask, DNS, DHCP)

**Path to IPv4 settings:** Network and Sharing Center → Click connection → Properties → Select "Internet Protocol Version 4 (TCP/IPv4)" → Properties

---

### System Applet

**Information shown:**
- Windows edition and version
- Processor (CPU)
- Installed RAM
- System type (32-bit/64-bit)

**Advanced System Settings (left sidebar):**
| Setting | What It Does |
|---------|--------------|
| Performance (Settings) | Visual effects, virtual memory |
| Virtual Memory | Paging file size configuration |

**Virtual Memory path:** Advanced tab → Performance Settings → Advanced tab → Virtual Memory

---

### Windows Defender Firewall

**Two network profiles:**

| Profile | When Used |
|---------|-----------|
| **Private network** | Home or work (trusted) |
| **Public network** | Coffee shop, airport, hotel (restrictive) |

**Left sidebar options:**
- Turn Windows Defender Firewall on or off
- Restore defaults
- Advanced settings (detailed rules)

> **Note:** Firewall is **enabled by default** – no action needed for basic protection.

---

### Mail Applet

**Important:** Only appears if a supported email client (e.g., Microsoft Outlook) is installed.

**Features:**
- Set up email accounts
- Modify data file locations
- Configure profiles

---

### Sound Applet

| Tab | Function |
|-----|----------|
| **Playback** | Configure speakers, set default playback device |
| **Recording** | Configure microphone(s), set default recording device |

---

### User Accounts

**What you can do:**
- Change account name
- Change password
- Change account icon
- Manage encryption certificates
- Change User Account Control (UAC) settings
- Manage all user accounts on the system

**Account types:**
- Standard User
- Administrator

---

### Device Manager

**Purpose:** View and manage device drivers for all hardware.

**Right-click options on a device:**
| Option | Function |
|--------|----------|
| Update driver | Install newer driver |
| Disable device | Turn off hardware |
| Uninstall device | Remove driver |
| Scan for hardware changes | Detect new/removed hardware |
| Properties | Detailed configuration |

**Device Properties tabs:**
| Tab | Information |
|-----|-------------|
| **General** | Device status (working properly?) |
| **Driver** | Driver date, version, files |
| **Details** | Hardware IDs, other technical info |
| **Events** | Driver history (updates, errors) |
| **Resources** | Memory ranges, IRQs, interrupts |

---

### Indexing Options

**Purpose:** Controls how Windows searches files quickly.

**What you can configure:**
- Which folders are indexed (Modify button)
- Which file types are indexed (Advanced → File Types tab)

> **How it works:** Windows pre-indexes files for fast search results.

---

### Administrative Tools / Windows Tools

**Common tools found here:**
- Computer Management
- Event Viewer
- Task Scheduler
- Registry Editor
- Performance Monitor
- Disk Management
- Services

---

### File Explorer Options

| Tab | Configuration Options |
|-----|----------------------|
| **General** | Browse folders, click behavior, privacy |
| **View** | Show/hide files (including hidden/system files) |
| **Search** | Include system directories, search compressed files (ZIP), search file names vs. content |

---

### Power Options

**Power plans:**
- Balanced (default)
- Power saver
- High performance

**Key settings:**
| Setting | Description |
|---------|-------------|
| **Sleep** | Keeps data in RAM (requires constant power) |
| **Hibernate** | Saves data to disk (no power needed, slower resume) |
| **Fast Startup** | Hybrid shutdown for faster boot (system never fully shuts down) |
| **Display timeout** | Turn off display after X minutes |
| **Sleep timeout** | Put computer to sleep after X minutes |
| **Lid closure action** | What happens when closing laptop lid |

**Advanced power settings:**
- Hard disk
- USB selective suspend
- PCI Express
- Processor power management

> **Exam distinction:** **Sleep** = RAM (needs power, fast resume). **Hibernate** = Disk (no power needed, slower resume).

---

### Ease of Access Center

**Accessibility features:**
| Feature | Function |
|---------|----------|
| Magnifier | Zoom screen content |
| On-Screen Keyboard | Type with mouse/touch |
| Narrator | Reads screen aloud |
| High Contrast | Improves visibility |

## 4. Sample CompTIA-Style Questions

> *A user needs to configure a proxy server for their browser. Which Control Panel applet and tab should they use?*
>
> A. Network and Sharing Center → Change adapter settings  
> B. Internet Options → Connections tab → LAN Settings  
> C. Internet Options → Advanced tab  
> D. Windows Defender Firewall → Advanced settings  

**Answer:** B

> *A technician wants to prevent a laptop from automatically putting the hard drive to sleep when running on battery. Where should they configure this?*
>
> A. Device Manager → Disk drives → Properties  
> B. Power Options → Advanced settings  
> C. System → Advanced System Settings  
> D. File Explorer Options → View tab  

**Answer:** B (Advanced power settings)

> *A user reports their microphone is not working. Which Control Panel applet should they use to check if it's properly configured?*
>
> A. Device Manager  
> B. Sound → Recording tab  
> C. System → Device properties  
> D. Ease of Access Center  

**Answer:** B (Sound applet, Recording tab)

> *Which Windows feature saves open documents and applications to the hard drive so the computer can be completely powered off without losing work?*
>
> A. Sleep  
> B. Fast Startup  
> C. Hibernate  
> D. Standby  

**Answer:** C (Hibernate saves to disk; no power needed)

> *A technician notices the Mail applet is missing from Control Panel. What is the most likely reason?*
>
> A. Windows is not activated  
> B. Microsoft Outlook is not installed  
> C. The user does not have administrator privileges  
> D. The applet is hidden by Group Policy  

**Answer:** B (Mail applet only appears when a supported email client like Outlook is installed)

## 5. Exam Tips Summary

| Scenario | Control Panel Applet |
|----------|---------------------|
| Change proxy settings | Internet Options → Connections |
| See what version of Windows is running | System |
| Check if a device driver is working | Device Manager |
| Change virtual memory (page file) size | System → Advanced System Settings → Performance |
| Stop a laptop from sleeping when lid closes | Power Options → Choose what closing the lid does |
| Add a new local user account | User Accounts → Manage another account |
| Turn off Windows Fast Startup | Power Options → Choose what power buttons do |
| Make Windows read screen content aloud | Ease of Access Center → Narrator |
| Change which folders Windows searches | Indexing Options → Modify |
| Uninstall an application | Programs and Features |
