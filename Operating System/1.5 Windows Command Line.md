## 1. Command Prompt Basics

### How to Launch

| Method | Command/Action |
|--------|----------------|
| Standard prompt | `cmd` (search or Run) |
| **Elevated (Admin)** | Right-click → Run as Administrator, or `Ctrl + Shift + Enter` |
| Why elevated? | Some utilities require admin rights (chkdsk, diskpart, format, sfc) |

> **Key point:** Standard privileges work for most utilities, but system changes require **elevated prompt**.

### Getting Help

| Method | Example | Result |
|--------|---------|--------|
| `help` | `help` | Lists all available commands |
| `help [command]` | `help dir` | Help specific to that command |
| `[command] /?` | `dir /?` | Same help information |

## 2. File and Directory Navigation Commands

| Command | Short Form | Purpose |
|---------|------------|---------|
| `dir` | `dir` | Display files/folders in current directory |
| `chdir` | `cd` | Change current working directory |
| `mkdir` | `md` | Create a new directory |
| `rmdir` | `rd` | Remove a directory |

### Navigation Examples

| Command | Effect |
|---------|--------|
| `cd Documents` | Move into Documents folder |
| `cd ..` | Move up one directory (parent folder) |
| `cd \` | Move to root of drive |
| `cd C:\Users\Professor\Documents` | Move to full path |

### Path Structure
- **Drive letter:** `C:` (primary drive)
- **Root:** `C:\`
- **Full path:** `C:\Users\Professor\Documents`
- **Delimiter:** Backslash (`\`)

### Tab key trick: Type partial name and press **Tab** to autocomplete.

## 3. Disk and File System Commands

### Check Disk (`chkdsk`)

| Option | Purpose | Time |
|--------|---------|------|
| `chkdsk /f` | Fix file system index errors | Faster |
| `chkdsk /r` | Sector-by-sector diagnostic + fix errors | Much slower |

**What `/r` does:**
- Checks every sector for read/write capability
- Recovers readable data
- **Includes all checks from `/f`**

> **Note:** If volume is in use (e.g., C: drive), chkdsk will schedule scan for next reboot.

### Format Command

| Command | Purpose |
|---------|---------|
| `format K:` | Formats drive K: with default (FAT32) |
| `format K: /FS:NTFS` | Formats with NTFS |

**Warning:** Format **completely erases** everything on the partition. Always have a backup.

### Disk Partition (`diskpart`)

| Command (inside diskpart) | Purpose |
|---------------------------|---------|
| `list volume` | Show all volumes with drive letters |
| `list disk` | Show physical disks |
| `list partition` | Show partitions on selected disk |
| `select volume X` | Choose a volume to work with |

**How to run:**
1. `diskpart` (requires admin)
2. UAC prompt appears
3. New admin command prompt opens

### Copy Commands

| Command | Features |
|---------|----------|
| `copy` | Basic file copy |
| `copy /v` | Verify copy (checks integrity) |
| `copy /y` | Suppress overwrite prompt (for automated scripts) |
| `robocopy` | **Robust Copy** – many advanced options |

**Robocopy advanced features:**
- Throttle bandwidth usage
- Retry on network interruption
- Copy subdirectories
- Resume interrupted copies

**Example:**
```
copy report.txt X:
copy /v report.txt X:    (verify)
copy /v /y report.txt X: (verify + auto-overwrite)
```

## 4. System Information Commands

| Command | Information Provided |
|---------|---------------------|
| `hostname` | Windows device name |
| `winver` | Windows version (10, 11, build number) |
| `whoami` | Current user (device\username) |
| `whoami /all` | User SID, groups, privileges, security info |

**Example outputs:**
- `hostname` → `Daedalus`
- `whoami` → `Daedalus\Professor`
- `whoami /all` → Shows SID, group memberships, privileges

## 5. Active Directory and Group Policy Commands

| Command | Purpose |
|---------|---------|
| `gpupdate` | Force Group Policy update (no reboot needed) |
| `gpupdate /force` | Reapply all policies (even unchanged ones) |
| `gpresult /r` | Show RSOP (Resultant Set of Policies) |

**What `gpresult /r` shows:**
- Domain name
- Logged-in user
- Computer name
- Windows version
- Applied Group Policy Objects (GPOs)
- Security groups
- Last policy application time

> **Use case:** After running `gpupdate`, run `gpresult /r` to verify new policy applied.

---

## 6. System File Checker (`sfc`)

| Command | Purpose |
|---------|---------|
| `sfc /scannow` | Scan all protected Windows files and repair corrupted ones |

**What it does:**
- Checks core Windows files for modifications
- Detects changes from third-party apps, updates, or malware
- Repairs corrupt or changed files

**Output example:** *"Windows Resource Protection found corrupt files and successfully repaired them."*

**Log location:** `cbs.log` (details of what was changed)

## 8. Sample CompTIA-Style Questions

> *A technician needs to repair corrupted Windows system files. Which command should they run from an elevated command prompt?*
>
> A. `chkdsk /f`  
> B. `sfc /scannow`  
> C. `gpupdate /force`  
> D. `diskpart`  

**Answer:** B (System File Checker repairs Windows system files)

> *A user reports their computer is part of an Active Directory domain, and a new desktop background policy isn't taking effect. Which command should they run to apply the policy without logging off?*
>
> A. `gpresult /r`  
> B. `whoami /all`  
> C. `gpupdate`  
> D. `winver`  

**Answer:** C (`gpupdate` forces Group Policy refresh)

> *Before shutting down a remote server, a technician wants to verify they are on the correct system. Which command should they use?*
>
> A. `winver`  
> B. `whoami`  
> C. `hostname`  
> D. `gpresult /r`  

**Answer:** C (`hostname` displays the computer name)

> *A technician needs to copy a large file across a slow WAN link and wants to ensure the copy completes even if the connection drops. Which command should they use?*
>
> A. `copy /v`  
> B. `copy /y`  
> C. `robocopy`  
> D. `xcopy`  

**Answer:** C (`robocopy` has retry and resume capabilities)

> *Which command would show a user's SID, group memberships, and security privileges?*
>
> A. `whoami`  
> B. `whoami /all`  
> C. `gpresult /r`  
> D. `winver`  

**Answer:** B (`whoami /all` provides complete security information)

## 9. Exam Tips Summary

| Scenario | Command |
|----------|---------|
| Need admin rights | Run `cmd` as Administrator / `Ctrl+Shift+Enter` |
| File system corrupted | `chkdsk /f` or `/r` |
| Drive failing (bad sectors) | `chkdsk /r` |
| Starting fresh on a drive | `format X:` |
| Creating/deleting partitions | `diskpart` |
| Copying with verification | `copy /v` |
| Automated script (no prompts) | `copy /y` |
| Large file over unstable network | `robocopy` |
| Identify current machine | `hostname` |
| Check Windows version | `winver` |
| Identify logged-in user | `whoami` |
| Force policy update | `gpupdate` |
| See applied policies | `gpresult /r` |
| Repair Windows system files | `sfc /scannow` |
