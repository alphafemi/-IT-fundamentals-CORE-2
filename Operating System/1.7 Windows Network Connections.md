## 1. Configuring New Network Connections

### Access Methods

| Method | Path |
|--------|------|
| **Control Panel** | Network and Sharing Center → Set up a new connection or network |
| **Settings App** | Settings → Network & Internet |

### Connection Types Available

| Connection Type | Description |
|----------------|-------------|
| **Ethernet** | Direct wired connection |
| **VPN** | Virtual Private Network (encrypted over public network) |
| **Dial-up** | Legacy telephone modem connection |
| **Wireless (Wi-Fi)** | SSID-based wireless connection |
| **WWAN** | Cellular (wireless wide area network) |

## 2. VPN (Virtual Private Network)

**Purpose:** Securely communicate through a public network using encryption.

### How VPN Works

| Component | Role |
|-----------|------|
| **VPN Client** | Software on your laptop that encrypts traffic before sending |
| **VPN Concentrator** | Corporate device that decrypts incoming traffic and encrypts outgoing |
| **Tunnel** | Encrypted path through public network (e.g., coffee shop Wi-Fi) |

### VPN Configuration in Windows

**Setup path:** Network and Sharing Center → Set up a new connection → Connect to a workplace → VPN

**Required information:**

| Setting | Example |
|---------|---------|
| Internet address | `vpn.example.com` (VPN concentrator) |
| Destination name | "VPN Connection" (friendly name) |
| Authentication | Password, smartcard, or both |

**Multi-factor authentication options:**
- Something you know (password)
- Something you have (smart card)
- Something you are (biometric/fingerprint)

> **Exam Tip:** Smart card integration adds "something you have" to authentication.

**How to connect:** Network Status icon (system tray) → Select VPN → Connect

## 3. Wireless (Wi-Fi) Configuration

### Key Wireless Terms

| Term | Definition |
|------|------------|
| **SSID** | Service Set Identifier – the name of the wireless network |
| **Security type** | Encryption protocol used (WPA2, WPA3, etc.) |
| **Encryption type** | TKIP (older) or AES (modern) |
| **PSK** | Pre-Shared Key (personal/home mode) |
| **802.1x** | Enterprise authentication (centralized) |

### Home vs. Enterprise Wireless

| Environment | Authentication | Key Type |
|-------------|----------------|----------|
| **Home/Personal** | WPA2/WPA3 Personal | PSK (Pre-Shared Key) |
| **Enterprise/Corporate** | WPA2/WPA3 Enterprise | 802.1x (central authentication server) |

> **Exam Tip:** 802.1x uses a central authentication server (RADIUS) – same credentials as corporate network.

## 4. Wired (Ethernet) Connections

**Characteristics:**
- Fastest connection type
- Default connection when multiple connections exist
- Plug-and-play (usually DHCP by default)

### Priority Rule
> *"The fastest connection is always the default connection."*

If connected to Ethernet + Wi-Fi + WWAN simultaneously, **Ethernet** takes priority.

### Ethernet Configuration Path

1. Right-click Ethernet adapter → Properties
2. Internet Protocol Version 4 (TCP/IPv4) → Properties
3. Options:
   - Obtain IP automatically (DHCP)
   - Use manual IP address (static)
   - Configure Alternate Configuration (fallback if DHCP unavailable)

## 5. Proxy Server Configuration

**Proxy role:** Intermediate device between client and internet (security, caching, filtering)

### Configuration Methods

| Method | Path |
|--------|------|
| **Control Panel** | Internet Options → Connections tab → LAN Settings |
| **Settings App** | Network & Internet → Proxy |

### Proxy Settings Options

| Setting | Description |
|---------|-------------|
| **Automatic configuration** | Windows detects proxy settings (WPAD) |
| **Manual proxy server** | Specify address and port manually |
| **Exceptions** | Applications that bypass proxy (connect directly) |

**Why exceptions?** Some applications don't work properly through a proxy.

## 6. Network Profiles

| Profile | Environment | Default Security |
|---------|-------------|------------------|
| **Private** | Home or work (trusted network) | Less restrictive – file sharing allowed |
| **Public** | Coffee shop, hotel, airport (untrusted) | Most restrictive – no file sharing, no inbound connections |

**Key difference:** Public network blocks inbound connections and disables file sharing by default.

> **Exam Tip:** Windows automatically assigns the profile based on network type (or asks the user on first connection).

## 7. Mapping Network Drives

### GUI Method (File Explorer)

**Steps:**
1. File Explorer → Three dots (or right-click This PC) → Map network drive
2. Select drive letter (e.g., H:)
3. Enter folder path: `\\server\share`
4. Check "Reconnect at sign-in" (optional)
5. Check "Connect using different credentials" (if needed)

**Example:** `\\Daedalus\Gate Room` (use quotes if spaces in share name)

### Command-Line Method (net use)

| Command | Purpose |
|---------|---------|
| `net use H: \\Daedalus\Gate Room` | Map drive (quotes if spaces: `"\\Daedalus\Gate Room"`) |
| `net use H: /delete` | Disconnect mapped drive |

### Disconnecting
- **GUI:** Right-click mapped drive → Disconnect
- **Command:** `net use H: /delete`

## 8. Metered Connections

**Purpose:** Reduce data usage when network charges by usage (cellular, hotspot, limited data plans)

### Settings Path

Settings → Network & Internet → Ethernet (or Wi-Fi) → **Metered connection** → On

### Advanced Data Limit Configuration

| Setting | Options |
|---------|---------|
| Limit type | Monthly, One-time, Unlimited |
| Reset date | When limit resets |
| Data limit | Specify GB/TB amount |

**What metered connection affects:**
- Windows Update downloads (reduced)
- App updates (limited)
- Background data usage (restricted)

---

## 9. WWAN (Wireless Wide Area Network)

**What it is:** Cellular network connectivity (4G/5G) for computers

**Connection methods:**

| Method | Description |
|--------|-------------|
| **Built-in WWAN card** | Internal cellular modem (like smartphone) |
| **USB WWAN adapter** | External plug-in device |
| **Phone tethering** | Connect phone via USB or Bluetooth |
| **Mobile hotspot** | Phone shares its cellular connection |

> **Note:** WWAN hardware may require **third-party software** – check manufacturer documentation.

## 10. Quick Reference: Network Connection Types

| Connection | Speed | Security | Configuration Location |
|------------|-------|----------|------------------------|
| **Ethernet** | Fastest | Physical | Adapter properties → IPv4 |
| **Wi-Fi** | Fast | WPA2/WPA3 | Wireless network settings |

## 11. Sample CompTIA-Style Questions

> *A user at a coffee shop wants to ensure maximum security for their laptop. Which network profile should Windows use?*
>
> A. Private  
> B. Domain  
> C. Public  
> D. Work  

**Answer:** C (Public profile – most restrictive, no file sharing, no inbound connections)

> *A technician is configuring a VPN connection for a remote employee. What information does the technician need from the corporate network?*
>
> A. SSID and PSK  
> B. Internet address of the VPN concentrator  
> C. Proxy server address and port  
> D. Ethernet MAC address  

**Answer:** B (The VPN concentrator's internet address – e.g., `vpn.company.com`)

> *Which command would map drive S: to a share named `\\server\documents`?*
>
> A. `net share S: \\server\documents`  
> B. `net use S: \\server\documents`  
> C. `map drive S: \\server\documents`  
> D. `net view S: \\server\documents`  

**Answer:** B (`net use` maps network drives)

> *A user has a laptop with Ethernet connected and Wi-Fi connected. Which connection will Windows use by default?*
>
> A. Wi-Fi (most secure)  
> B. Ethernet (fastest)  
> C. Both simultaneously (load balancing)  
> D. User must manually select  

**Answer:** B (Windows uses the fastest connection – typically Ethernet)

> *A corporate network uses WPA2-Enterprise with 802.1x. What does this require?*
>
> A. A pre-shared key (PSK) on all devices  
> B. A central authentication server (RADIUS)  
> C. TKIP encryption only  
> D. No authentication required  

**Answer:** B (802.1x uses a central authentication server for enterprise credentials)

> *A user is traveling and paying for hotel Wi-Fi by the gigabyte. What should they enable to reduce data usage?*
>
> A. VPN connection  
> B. Metered connection  
> C. Public network profile  
> D. Proxy server  

**Answer:** B (Metered connection reduces data usage)

## 12. Exam Tips Summary

| Scenario | Solution/Knowledge |
|----------|-------------------|
| Secure connection over public Wi-Fi | Use **VPN** (encrypted tunnel) |
| Coffee shop security | **Public** network profile |
| Home network file sharing | **Private** network profile |
| Limited data plan | Enable **Metered connection** |
| Access network share as local drive | **Map network drive** (net use) |
| Central wireless authentication for business | **WPA2/WPA3 Enterprise** with 802.1x |
| Home wireless authentication | **WPA2/WPA3 Personal** with PSK |
| Application won't work through proxy | Add **exception** |
| Wi-Fi uses AES encryption | Modern standard (not TKIP) |
| Connect via cellular | **WWAN** (Wireless Wide Area Network) |
