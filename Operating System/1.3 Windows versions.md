A clearly outlined  editions, features, and lifecycle differences between Windows 10 and Windows 11, as well as the specific terminology (like the "N edition") that the exam loves to test.



---

### 1. 
- **In-Scope Versions:** All in-support Windows versions. You need to know **Windows 10** and **Windows 11**.
- **The "Windows 9" Gap:** Microsoft skipped Windows 9 entirely (went from 8 to 10).
- **Windows 10 EOL (End of Life):** October 14, 2025 (for Home and Pro).
    - *Important:* CompTIA expects you to know this, but also knows organizations use OSes past their lifespan.
- **Windows 11 Release:** October 2021.

---

### 2. Windows 10 Editions & Features
*Note: There were 15+ feature releases (final was 21H2).*

| Feature | Home | Pro | Pro for Workstations | Enterprise |
| :--- | :---: | :---: | :---: | :---: |
| **Target User** | Consumer | Business | High-end hardware | Large org (VL) |
| **Domain Access** | No | Yes | Yes | Yes |
| **BitLocker (FDE)** | No | Yes | Yes | Yes |
| **Remote Desktop Host** | No | Yes | Yes | Yes |
| **Group Policy** | No | Yes | Yes | Yes |
| **Max RAM (64-bit)** | 128 GB | 2 TB | 6 TB | 6 TB |
| **Special Features** | OneDrive, Defender, Cortana | — | 4x physical CPUs, ReFS | AppLocker, BranchCache, UX control |

> **Key Exam Trick:** Only the **Home** edition lacks Domain Access, BitLocker, and Remote Desktop *host* capabilities (it can still be a client).

---

### 3. Windows 11 Editions & Features
*Key change: No 32-bit CPU support (64-bit only).*

| Feature | Home | Pro | Enterprise |
| :--- | :---: | :---: | :---: |
| **Target User** | Consumer | Business | Large org (VL) |
| **Domain Access** | No | Yes | Yes |
| **BitLocker** | No (uses Device Encryption) | Yes | Yes |
| **Remote Desktop Host** | No | Yes | Yes |
| **Group Policy** | No | Yes | Yes |
| **Max RAM** | 128 GB | 2 TB | 6 TB |
| **Unique Features** | MS Account integration, Device Encryption | Hyper-V | MDM, MAM, ReFS |

**Windows 11 Specifics to Memorize:**
- **Windows Copilot:** Built-in AI.
- **Snap Layouts:** New window management.
- **Teams Integration:** Built into the taskbar.
- **Device Encryption (Home only):** Like BitLocker, but recovery key saves to **Microsoft Account** (not Active Directory).

---

### 4. The "N Edition" (European Union)
- **What it is:** Windows without media player functionality.
- **Missing:** Windows Media Player and other multimedia utilities.
- **How to fix:** Install the **Media Feature Pack** (Settings > Apps > Optional Features).
- **Why it exists:** EU antitrust regulations.

---

### 5. High-End & Enterprise Specifics

#### Windows 10/11 Pro for Workstations
- **Hardware:** Supports up to **4 physical CPUs**.
- **Memory:** Up to **6 TB RAM**.
- **File System:** Supports **ReFS** (Resilient File System).

#### Windows 10/11 Enterprise
- **Licensing:** Volume Licensing (VL).
- **AppLocker:** Restrict which applications run.
- **BranchCache:** Cache data at remote sites (saves WAN bandwidth).
- **MDM/MAM:** Mobile Device/Application Management (Windows 11).
- **UX Control:** Granular control over what users see.

### 7. CompTIA Core 2 Exam Tips 

1.  **Domain vs. Workgroup:**
    - *Home:* Cannot join a domain.
    - *Pro/Enterprise:* Can join a domain.
2.  **Remote Desktop:**
    - *Home:* Client only (can connect *to* others).
    - *Pro/Enterprise:* Host & Client (can be connected *to*).
3.  **BitLocker vs. Device Encryption:**
    - *BitLocker:* Pro/Enterprise. Recovery in AD.
    - *Device Encryption:* Home (Windows 11). Recovery in MS Account.
4.  **Memory Limits (64-bit):**
    - Home: 128 GB (Win10/11).
    - Pro: 2 TB.
    - Enterprise/Workstations: 6 TB.
5.  **ReFS (Resilient File System):**
    - Found in: Windows Server, **Pro for Workstations**, **Enterprise**.
    - *Not* in: Standard Pro or Home.
6.  **The "N" Edition:**
    - No media player. Fix = Media Feature Pack.

### Sample Exam Question ()
> *A user in the EU purchases a new laptop with Windows 11 Pro N installed. They try to play an MP4 video file, but nothing happens. What is the most likely reason?*
> A. BitLocker is encrypting the file.
> B. The N edition lacks media player functionality.
> C. The laptop has a 32-bit CPU.
> D. Remote Desktop Host is disabled.

**Answer:** B. (The N edition excludes Windows Media Player.)
