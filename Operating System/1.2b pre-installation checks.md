## 1. Upgrade vs. Clean Install: Core Differences

| Aspect | **In-Place Upgrade** | **Clean Install** |
|--------|---------------------|-------------------|
| **Starting point** | From inside existing OS | Boot from installation media |
| **Applications** | Preserved | Removed (must reinstall) |
| **User data** | Preserved | Removed (backup required) |
| **User settings** | Preserved | Removed |
| **Local user accounts** | Preserved | Removed |
| **Operating system files** | Replaced/updated | Completely new |
| **Time** | Faster (no reinstall of apps/data) | Slower (full reinstall + restore) |
| **Best for** | Multiple users, specialized configs, saving time | Problematic systems, security refresh, changing partition schemes |

> **Key quote:** *"Neither one is better than the other, but you should use the right one for your specific circumstance."*

## 2. Pre-Installation Checklist (Before You Start)

| Step | Why It Matters |
|------|----------------|
| **Check drive contents** | Is there data? Other partitions the user isn't aware of? |
| **Backup everything** | Even if user says "I don't need anything" – recovery is impossible after repartition/format |
| **Save user preferences** | Can be copied back after clean install |
| **Verify OS requirements** | Memory, drive space, processor, feature support |
| **Run compatibility check** | Microsoft's PC Health Check (Windows 11) |
| **Check app/driver compatibility** | Some apps or drivers may not work on newer OS |
| **Have license keys ready** | Installation may require them |
| **Know your answers** | Which drive? Partition scheme? Configuration choices? |

## 3. Windows 11 Specific Requirements

Windows 11 has stricter hardware requirements than previous versions. Your text highlights these key checks:

### Required Components

| Requirement | Tool to Check | What to Look For |
|-------------|---------------|------------------|
| **TPM 2.0** | `tpm.msc` | Trusted Platform Module version 2.0 or later |
| **Secure Boot** | System Information (`msinfo32`) → System Summary → Secure Boot State | Must be **On** |
| **UEFI BIOS** | System Information | Must be UEFI, not legacy BIOS |
| **Processor support** | PC Health Check | Listed as supported |
| **Sufficient RAM** | PC Health Check / System Properties | Meets Windows 11 minimum |

### PC Health Check for Windows 11
- Run before attempting upgrade
- Provides pass/fail for each requirement
- Example from your text: *"This PC doesn't currently meet the Windows 11 system requirements"* – specifically failed on Secure Boot and TPM 2.0, even though processor and memory were sufficient.

> **Important:** If a computer lacks UEFI BIOS (older legacy BIOS only), you may need to **physically replace the computer** – it cannot run Windows 11.

## 4. What is TPM (Trusted Platform Module)?

| Feature | Description |
|---------|-------------|
| **What it is** | Cryptographic hardware (usually on motherboard) |
| **Required version for Windows 11** | TPM 2.0 or later |
| **Used by** | BitLocker, Windows Hello, and other security features |
| **How to check** | Run `tpm.msc` (Microsoft Management Console snap-in) |

## 5. Operating System Lifecycle

Most OS manufacturers provide a **lifecycle calendar** that includes:

| Update Type | Frequency | Purpose |
|-------------|-----------|---------|
| **Quality updates** | Monthly | Security patches, bug fixes |
| **Feature updates** | Every 6–12 months | New capabilities |

**Typical support duration:** 18–36 months (varies by Windows version and edition)

**Microsoft's policy:** Referred to as the **Modern Lifecycle Policy** – details on Microsoft website.

## 6. Quick Decision Guide

| Scenario | Recommended Action |
|----------|-------------------|
| Many local user accounts, specialized configs | In-place upgrade |
| Computer running slowly, malware issues, or you want a fresh start | Clean install (after backup) |
| Moving from Windows 10 to Windows 11 | First run PC Health Check; verify TPM 2.0 + Secure Boot |
| Older computer (no UEFI, no TPM 2.0) | Replace hardware; cannot upgrade to Windows 11 |
| You forgot to check drive contents before clean install | STOP – backup first, then proceed |
| App or driver not compatible with new OS | Stay on current OS or find updated versions |
